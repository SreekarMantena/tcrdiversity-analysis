This subfolder contains scripts used to process alleles.

assembly_map.sh maps the assembly sequences to the T2T reference TCR loci.
read_support.sh quantifies the read support for candidate alleles.
allele_viz.r generates summary discovery plots of the alleles.
site_properties.r generates plots of sites across alleles.
functionality.r generates plots of nonfunctional alleles.
IMGT_GENEDB_250611_tcrvdjregions.fasta.zip contains the version of the IMGT database that was used in the study, downloaded on 06/11/25.
